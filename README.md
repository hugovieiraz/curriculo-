# Portfólio — Victor Hugo Vieira Tavares

Site pessoal/portfólio acadêmico, estudante de Engenharia Elétrica na UFCG com foco em eletrônica de potência.

Site estático (HTML + CSS, sem build). Para visualizar localmente, basta abrir o `index.html` no navegador.

## Estrutura

```
.
├── index.html      # página única
├── images/         # fotos e imagens
├── .nojekyll       # serve os arquivos como estão no GitHub Pages
└── README.md
```

## Publicação no GitHub Pages

Após o push para o repositório, ative em **Settings → Pages → Branch: `main` / root**.

O site fica disponível em: `https://hugovieiraz.github.io/curriculo/`
